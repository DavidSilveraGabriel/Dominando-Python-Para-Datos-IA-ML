Introducción a los IDEs

Herramientas esenciales para escribir, ejecutar y depurar código

¿Qué es un IDE?

Un Entorno de Desarrollo Integrado proporciona herramientas para facilitar el desarrollo de software:

Resaltado de sintaxis

Autocompletado

Depuración

Integración con Git

Gestión de proyectos

Visual Studio Code

Editor de código fuente gratuito, ligero pero potente, desarrollado por Microsoft.

Fortalezas:

  • Excelente para scripts, módulos y proyectos completos
  • Gran soporte para Python a través de extensiones
  • Terminal integrada y depuración robusta
  • Soporta Jupyter Notebooks dentro del editor

Jupyter Notebook / Lab

Aplicación web que permite crear documentos con código vivo, visualizaciones y texto narrativo.

Fortalezas:

  • Ideal para exploración de datos interactiva y visualización
  • Ejecuta código en celdas individuales con resultados inmediatos
  • Excelente para documentar el proceso de análisis
  • Se integra perfectamente con Pandas, Matplotlib, etc.

Instalación

VS Code

Descarga desde code.visualstudio.com e instala la extensión "Python" de Microsoft

Jupyter

  • • Incluido en Anaconda
  • • Con Miniconda: conda install jupyterlab notebook
  • • Iniciar: jupyter lab o jupyter notebook
