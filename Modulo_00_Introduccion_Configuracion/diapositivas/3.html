Instalación de Python y Anaconda

Configurando el entorno ideal para Ciencia de Datos e IA

¿Por qué Anaconda/Miniconda?

  • Incluye Python instalado y listo para usar
  • Gestor de paquetes conda para instalación y gestión de librerías
  • Crea entornos virtuales aislados para proyectos independientes
  • Maneja dependencias complejas del ecosistema científico

Anaconda vs Miniconda

Anaconda

Distribución completa con Python, conda y cientos de librerías preinstaladas

Incluye NumPy, Pandas, Matplotlib, Scikit-learn, Jupyter, etc.

Ideal si tienes espacio en disco (~3GB)

Todo listo desde el principio

Miniconda

Versión mínima con solo Python, conda y dependencias básicas

Instalas manualmente las librerías que necesitas

Instalación más ligera (~400MB)

Más control sobre qué paquetes se instalan

Recomendación para el curso

Si eres principiante y tienes espacio, Anaconda es más sencillo. Si prefieres control total o tienes espacio limitado, Miniconda es excelente (instalarás librerías a medida que las necesites con conda install <nombre_libreria>).
