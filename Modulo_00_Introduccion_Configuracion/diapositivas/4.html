  • Ejecutar comandos de Python, conda, git
  • Navegar entre directorios eficientemente
  • Gestión de archivos y carpetas
  • Automatización de tareas repetitivas

Comandos Básicos

Navegación

cd <dir> Cambiar directorio
cd .. Subir un nivel
ls / dir Listar contenido
pwd / cd Mostrar ubicación

Gestión

mkdir <dir> Crear carpeta
touch / type nul > Crear archivo
cp / copy Copiar
mv / move Mover/Renombrar

Python y Más

python <script.py> Ejecutar script
pip install <pkg> Instalar paquete
clear / cls Limpiar pantalla
Tab Autocompletar

Ejemplo Práctico

cd Desktop
mkdir CursoPython
cd CursoPython
