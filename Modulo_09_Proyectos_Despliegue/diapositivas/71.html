Creación de APIs Simples con Flask/FastAPI

Exponiendo tus modelos y análisis como servicios web.

Flask

¿Qué es? Un microframework web simple y extensible.

Ventajas: Fácil de aprender, flexible, gran comunidad.

Ideal para: APIs pequeñas/medianas, prototipos rápidos.

Ejemplo Básico:

# app_flask.py
from flask import Flask, request, jsonify

app = Flask(__name__)

@app.route('/predict', methods=['POST'])
def predict():
    data = request.get_json()
    features = data.get('features')
    # Simular predicción
    prediction = sum(features) > 10
    return jsonify({'prediction': int(prediction)})

if __name__ == '__main__':
    app.run(debug=True, port=5000)

FastAPI

¿Qué es? Framework web moderno de alto rendimiento.

Ventajas: Muy rápido, fácil de codificar, validación automática (Pydantic), documentación automática (Swagger UI).

Ideal para: Nuevas APIs, alto rendimiento, robustez del tipado.

Ejemplo Básico:

# app_fastapi.py
from fastapi import FastAPI
from pydantic import BaseModel
from typing import List

app = FastAPI()

class InputFeatures(BaseModel):
    features: List[float]

@app.post("/predict")
async def predict(input_data: InputFeatures):
    features = input_data.features
    # Simular predicción
    prediction = sum(features) > 10
    return {'prediction': int(prediction)}

# Ejecutar con: uvicorn app_fastapi:app --reload

Ambos son excelentes opciones. La elección depende de tus necesidades específicas.
