Módulo 9: Desarrollo Guiado de Proyectos de Portafolio

Aplicando tus conocimientos en proyectos prácticos.

¡Es Hora de Construir!

Este módulo te guiará en la aplicación de todo lo aprendido en proyectos reales.

El objetivo es crear proyectos para tu portafolio profesional, integrando Python, NumPy, Pandas, Visualización y Machine Learning.

Proyecto 1: Análisis Exploratorio de Datos (EDA)

Aplicar limpieza, manipulación, agregación y visualización a un dataset público.

Ejemplos: Titanic, Precios de Viviendas, Películas.

Proyecto 2: Modelo Predictivo Simple

Construir un modelo básico de ML (Clasificación o Regresión) con Scikit-learn.

Siguiendo el flujo de trabajo: preparación, división, entrenamiento, predicción, evaluación.

Proyecto 3: API Simple o Dashboard (Opcional)

Exponer un modelo vía API (Flask/FastAPI) o crear un dashboard interactivo (Streamlit/Plotly Dash).

Un paso más para mostrar tus proyectos.

Estos proyectos te ayudarán a consolidar habilidades y destacar en tu portafolio.
