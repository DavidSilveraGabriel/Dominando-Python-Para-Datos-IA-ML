Módulo 3: Creación, Indexación y Slicing

Creación de Arrays

np.array(lista): Desde lista/tupla.

np.zeros(shape): Lleno de ceros.

np.ones(shape): Lleno de unos.

np.full(shape, val): Lleno de `val`.

np.empty(shape): Sin inicializar.

np.arange(start, stop, step): Rango (como `range`).

np.linspace(start, stop, num): `num` puntos entre `start` y `stop`.

np.identity(n) / np.eye(n): Matriz identidad.

np.random.rand(d0...): Aleatorios [0, 1).

np.random.randn(d0...): Aleatorios Normal(0,1).

np.random.randint(low, high, size): Enteros aleatorios.

z = np.zeros((2,2))
a = np.arange(0, 5, 2) # [0 2 4]
r = np.random.rand(2)

Indexación (Acceso)

Índices basados en cero.

Arrays 1D:

arr1d = np.arange(10, 15)
# [10 11 12 13 14]
print(arr1d[0])    # 10
print(arr1d[-1])   # 14
arr1d[0] = 100   # Modificar

Arrays 2D (Matrices):

arr2d = np.array([[1,2],[3,4]])
print(arr2d[0, 0]) # 1 (fila 0, col 0)
print(arr2d[1, 0]) # 3 (fila 1, col 0)
# Alt: arr2d[1][0]
arr2d[1, 1] = 44 # Modificar

Slicing (Rebanado)

Seleccionar subconjuntos (`start:stop:step`).

Arrays 1D:

arr = np.arange(5) # [0 1 2 3 4]
print(arr[:3])   # [0 1 2]
print(arr[2:])   # [2 3 4]
print(arr[1:4])  # [1 2 3]
print(arr[::2])  # [0 2 4]
print(arr[::-1]) # [4 3 2 1 0]

Arrays 2D:

arr2d = np.arange(6).reshape(2,3)
# [[0 1 2], [3 4 5]]
# arr[filas, columnas]
print(arr2d[0, :]) # Fila 0 -> [0 1 2]
print(arr2d[:, 1]) # Col 1 -> [1 4]
print(arr2d[:1, 1:])# Fila 0, cols 1,2 -> [[1 2]]

Vistas vs Copias

Slicing crea vistas (referencias) por defecto. Modificar la vista modifica el original.

arr = np.arange(3)
vista = arr[1:] # [1 2]
vista[0] = 99
# arr ahora es [0 99 2] !

Usa .copy() para una copia independiente: copia = arr[1:].copy()

Indexación Booleana

Seleccionar elementos con condiciones.

arr = np.array([2, 5, 1, 8])
cond = arr > 3 # [False True False True]
print(arr[cond]) # [5 8]
# Combinar: arr[(arr>3) & (arr<8)] -> [5]
arr[arr < 5] = 0 # Modificar -> [0 5 0 8]
