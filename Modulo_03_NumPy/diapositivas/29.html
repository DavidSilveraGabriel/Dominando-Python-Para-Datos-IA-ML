Módulo 3: Operaciones Matemáticas y Estadísticas Vectorizadas

La magia de NumPy: operaciones rápidas sin bucles `for` explícitos.

Aritmética Elemento a Elemento

Operadores (`+`, `-`, `*`, `/`, `**`, etc.) entre arrays compatibles o array-escalar.

a = np.array([1, 2])
b = np.array([10, 20])
print(a + b)    # [11 22]
print(a * 3)    # [3 6]
print(a ** 2)   # [1 4]
print(a > 1)    # [False True]

¡Mucho más rápido que bucles `for` en Python!

Funciones Universales (ufuncs)

Funciones optimizadas (en C) que operan elemento a elemento.

Unarias (1 array):

np.sqrt, np.exp, np.log, np.sin, np.abs, np.square, np.ceil, np.floor, np.isnan, ...

Binarias (2 arrays):

np.add, np.subtract, np.multiply, np.divide, np.power, np.maximum, np.greater, ...

arr = np.array([1, 4, 9])
print(np.sqrt(arr)) # [1. 2. 3.]

a = np.array([1, 5])
b = np.array([2, 3])
print(np.maximum(a, b)) # [2 5]

Métodos Estadísticos

Métodos del array para cálculos agregados.

.sum(), .mean(), .std(), .var(), .min(), .max(), .argmin(), .argmax(), .cumsum(), ...

Parámetro `axis`:

axis=0 (columnas), axis=1 (filas).

m = np.array([[1,2],[3,4]])
print(m.sum())      # 10 (total)
print(m.mean())     # 2.5 (total)
print(m.sum(axis=0))# [4 6] (sum cols)
print(m.mean(axis=1))#[1.5 3.5](mean rows)

Métodos Booleanos

.any(): ¿Al menos un `True`?
.all(): ¿Todos `True`?

b = np.array([True, False])
print(b.any()) # True
print(b.all()) # False
print((m > 0).all()) # True
