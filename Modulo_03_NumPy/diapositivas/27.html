Módulo 3: Introducción a NumPy y Arrays (`ndarray`)

La base de la computación científica en Python.

¿Qué es NumPy?

Biblioteca fundamental para computación científica:

  • Objeto **`ndarray`** (array N-dimensional) eficiente.
  • Operaciones matemáticas rápidas sobre arrays.
  • Álgebra lineal, FFT, números aleatorios.
  • Integración con C/C++/Fortran.
  • Base para Pandas, Scikit-learn, etc.

¿Por qué NumPy vs Listas?

  • Eficiencia: Más rápidos y menor uso de memoria para números.
  • Conveniencia: Operaciones vectorizadas (sin bucles `for` explícitos).
  • Funcionalidad: Amplia gama de funciones matemáticas optimizadas.

Importando NumPy

La convención universal es usar el alias `np`.

import numpy as np

(Instalar si es necesario: `pip install numpy` o `conda install numpy`)

El `ndarray` (N-dimensional array)

Cuadrícula multidimensional de elementos del mismo tipo y tamaño fijo.

Características Clave:

  • Multidimensional: Vectores (1D), matrices (2D), tensores (3D+).
  • Homogéneo: Todos los elementos del mismo `dtype`.
  • Tamaño Fijo: Definido al crear el array.
  • Indexado: Acceso por índices `[0]`, `[fila, col]`, etc.

Creación Básica y Atributos:

import numpy as np

lista = [1, 2, 3]
arr1d = np.array(lista)
print(arr1d)       # [1 2 3]
print(arr1d.dtype) # int64 (o int32)
print(arr1d.ndim)  # 1
print(arr1d.shape) # (3,)
print(arr1d.size)  # 3

lista_anidada = [[1, 2], [3, 4]]
arr2d = np.array(lista_anidada)
print(arr2d.shape) # (2, 2)
print(arr2d.ndim)  # 2

Otros Atributos:

  • `ndarray.itemsize`: Tamaño en bytes de cada elemento.
  • `ndarray.data`: Buffer de memoria (raramente usado directamente).
