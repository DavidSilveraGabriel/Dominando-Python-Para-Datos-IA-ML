Módulo 8: Generadores

Iteración eficiente y perezosa.

¿Qué son los Generadores?

Son un tipo especial de **iterador** en Python. Permiten iterar sobre una secuencia de valores, pero a diferencia de las listas o tuplas, **no almacenan toda la secuencia en memoria** a la vez.

Generan los valores **sobre la marcha** (de forma "perezosa" o "lazy") a medida que se solicitan.

¿Por Qué Usar Generadores?

  • **Eficiencia de Memoria:** Ideales para trabajar con secuencias muy grandes o infinitas, ya que solo mantienen un elemento en memoria a la vez.
  • **Rendimiento:** Pueden ser más rápidos que construir listas completas para iteraciones únicas.
  • **Simplicidad:** El código a menudo es más limpio y legible que usar iteradores manuales.

Creando Generadores (`yield`)

Se crean usando **funciones generadoras**. Una función se convierte en generadora si contiene la palabra clave `yield`.

Cuando `yield` es ejecutado, la función pausa su ejecución y devuelve el valor especificado. La próxima vez que se llame al generador, la función retoma su ejecución desde donde se quedó.

Ejemplo de Función Generadora 
def contador_simple(n):
    i = 0
    while i < n:
        yield i # Pausa y devuelve i
        i += 1

# Usar el generador
mi_generador = contador_simple(5)

print(next(mi_generador)) # Salida: 0
print(next(mi_generador)) # Salida: 1

# O iterar con un bucle for
for num in contador_simple(3):
    print(num) # Salida: 0, 1, 2

Los generadores son una forma elegante y eficiente de manejar secuencias de datos.
