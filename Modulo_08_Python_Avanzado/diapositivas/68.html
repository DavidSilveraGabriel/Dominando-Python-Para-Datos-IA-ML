Módulo 8: Tipado Estático (Type Hinting)

Mejorando la claridad y detectando errores tempranamente.

¿Qué es el Tipado Estático en Python?

Python es dinámicamente tipado (el tipo se determina en tiempo de ejecución).

Las **Type Hints** (desde Python 3.5) permiten **anotar** tipos esperados para variables, parámetros y retornos.

¡Importante! Las Type Hints son opcionales y no son verificadas por el intérprete en tiempo de ejecución por defecto.

Su propósito principal es mejorar la legibilidad, documentación y permitir el análisis estático con herramientas externas (ej. MyPy).

Sintaxis Básica

Usa : tipo para variables y parámetros, y -> tipo para retornos.

Ejemplo 
# Variables
edad: int = 30
nombre: str = "Alice"

# Funciones
def saludar(nombre: str) -> None:
    print(f"Hola, {nombre}")

def sumar(a: int, b: int) -> int:
    return a + b

def procesar_datos(datos: list[float]) -> float:
    if not datos:
        return 0.0
    return sum(datos) / len(datos)

Tipos Complejos (`typing`)

Usa el módulo typing para listas, diccionarios, opcionales, uniones, etc.

Desde Python 3.9+, puedes usar tipos genéricos incorporados (ej. list[int]).

Ejemplo 
from typing import List, Dict, Optional, Union, Any, Callable

nombres: List[str] = ["Ana", "Luis"]
edades: Dict[str, int] = {"Ana": 30}
coordenada: tuple[float, float] = (10.5, -3.2) # Python 3.9+

def buscar_usuario(user_id: int) -> Optional[str]:
    if user_id == 1: return "Admin"
    return None

identificador: Union[int, str] = "ID-123"

def aplicar(func: Callable[[int, int], int], x: int, y: int) -> int:
    return func(x, y)

# Type Aliases
Vector = List[float]
def norma(v: Vector) -> float:
    return sum(x*x for x in v)**0.5

Verificación Estática con MyPy

Herramientas como **MyPy** analizan tu código *antes* de ejecutarlo para encontrar errores de tipo.

Instala: pip install mypy

Ejecuta: mypy tu_archivo.py

Ejemplo de Error Detectado 
# archivo_con_error_tipo.py
def duplicar(numero: int) -> int:
    return numero * 2

resultado_error = duplicar("hola") # MyPy detectará esto!

El tipado estático mejora la mantenibilidad y robustez del código, especialmente en proyectos grandes.
