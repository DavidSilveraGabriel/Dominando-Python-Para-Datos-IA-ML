Módulo 2: Introducción a Decoradores

Añadiendo funcionalidad extra a funciones existentes.

¿Qué es un Decorador?

Una función que recibe otra función, le añade "algo" (wrapper), y devuelve la función modificada.

Usos Comunes:

  • Logging, Temporización
  • Control de acceso, Validación
  • Caching, Instrumentación

Base: Funciones 1ª Clase

En Python, las funciones pueden ser:

  • Asignadas a variables
  • Pasadas como argumentos
  • Devueltas por otras funciones
  • Definidas dentro de otras (anidadas)

`*args` y `**kwargs`

Permiten que la función `wrapper` del decorador acepte cualquier argumento y lo pase a la función original, haciéndolo genérico.

  • `*args`: Captura argumentos posicionales en una tupla.
  • `**kwargs`: Captura argumentos de palabra clave en un dict.

Estructura Típica

def mi_decorador(func_original):
    def wrapper(*args, **kwargs): # 1. Envoltura genérica
        # 2. Código ANTES de llamar
        print("Antes...")
        # 3. Llamar a la original pasando args/kwargs
        resultado = func_original(*args, **kwargs)
        # 4. Código DESPUÉS de llamar
        print("Después...")
        # 5. Devolver resultado original
        return resultado
    return wrapper # 6. Devolver la envoltura

Aplicación con `@` (Azúcar Sintáctico)

@mi_decorador # <-- Aplica el decorador
def funcion_a_decorar(x):
    print(f"Ejecutando con {x}")
    return x * 10

# Es equivalente a:
# def funcion_a_decorar(x): ...
# funcion_a_decorar = mi_decorador(funcion_a_decorar)

valor = funcion_a_decorar(5)
print(f"Valor final: {valor}")

Al llamar a `funcion_a_decorar(5)`, en realidad se ejecuta `wrapper(5)`, que a su vez llama a la `funcion_a_decorar` original.

Salida: Antes... Ejecutando con 5 Después... Valor final: 50
