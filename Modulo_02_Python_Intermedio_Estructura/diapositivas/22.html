Módulo 2: Comprensión de Listas, Diccionarios y Conjuntos

Creando colecciones de forma concisa y Pythonica.

Listas (`[]`)

Sintaxis:
nueva = [expr for e in iter]
nueva = [expr for e in iter if cond]
nueva = [exprT if cond else exprF for e in iter]
Ejemplo:
# Cuadrados de pares
nums = range(6) # 0 a 5
cuad_pares = [x**2 for x in nums if x % 2 == 0]
# Resultado: [0, 4, 16]
print(cuad_pares)

# Par/Impar
par_impar = ["Par" if n%2==0 else "Impar" for n in nums]
# Resultado: ['Par','Impar','Par','Impar','Par','Impar']
print(par_impar)

Diccionarios (`{}`)

Sintaxis:
nuevo = {k_expr: v_expr for e in iter}
nuevo = {k_expr: v_expr for e in iter if cond}
Ejemplo:
# Número -> Cuadrado
cuadrados = {n: n**2 for n in range(4)}
# Resultado: {0: 0, 1: 1, 2: 4, 3: 9}
print(cuadrados)

# Palabra -> Longitud (si > 3)
palabras = ["sol", "luna", "estrella"]
longitudes = {p: len(p) for p in palabras if len(p)>3}
# Resultado: {'luna': 4, 'estrella': 8}
print(longitudes)

Conjuntos (`{}`)

Sintaxis:
nuevo = {expr for e in iter}
nuevo = {expr for e in iter if cond}
Ejemplo:
# Iniciales únicas (mayúsculas)
nombres = ["ana", "luis", "ana"]
iniciales = {nombre[0].upper() for nombre in nombres}
# Resultado: {'A', 'L'}
print(iniciales)

# Divisibles por 3 (0-10)
div3 = {n for n in range(11) if n % 3 == 0}
# Resultado: {0, 3, 6, 9}
print(div3)

Ventajas

Concisión, legibilidad (a menudo) y eficiencia.

¿Cuándo usar?

Ideal para crear colecciones a partir de transformaciones/filtrados simples. Si la lógica es muy compleja, un bucle `for` puede ser mejor.
