Módulo 2: Programación Orientada a Objetos (Clases y Objetos)

Modelando el mundo real con código.

Conceptos Clave

Clase (`class`)

El plano o plantilla para crear objetos.

Objeto (`object`)

La instancia concreta creada a partir de una clase.

Atributo (`attribute`)

Variable de la clase/objeto (característica).

Método (`method`)

Función de la clase (comportamiento).

`self`

Referencia al propio objeto (instancia) dentro de la clase.

`__init__(self, ...)`

Método constructor (inicializador), se llama al crear el objeto.

Definiendo una Clase

class NombreClase: # PascalCase
    """Docstring de la clase."""
    atributo_clase = "Compartido"

    def __init__(self, param1):
        # Constructor: Inicializa atributos de instancia
        self.atributo_instancia1 = param1
        self.otro_atributo = "Valor Fijo"

    def nombre_metodo(self, arg1):
        # Método de instancia (requiere 'self')
        print(f"Accediendo a {self.atributo_instancia1}")
        return arg1 * 2

Creando y Usando Objetos

# Instanciación (Llamar a la clase)
objeto1 = NombreClase("Valor para param1")
objeto2 = NombreClase("Otro valor")

# Acceder a atributos (notación de punto)
print(objeto1.atributo_instancia1) # Salida: Valor para param1
print(objeto2.atributo_instancia1) # Salida: Otro valor
print(objeto1.otro_atributo)      # Salida: Valor Fijo
print(NombreClase.atributo_clase) # Acceso al atributo de clase

# Llamar a métodos (notación de punto)
resultado = objeto1.nombre_metodo(10) # Pasa arg1=10 (self se pasa auto)
print(resultado) # Salida: 20
