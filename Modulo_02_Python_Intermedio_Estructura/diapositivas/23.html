Módulo 2: Funciones Lambda, `map` y `filter`

Herramientas para programación funcional y procesamiento de secuencias.

Lambda (Anónimas)

Funciones pequeñas, sin nombre, de una sola expresión.

Sintaxis:
lambda argumentos: expresion
Ejemplo:
sumar = lambda x, y: x + y
print(sumar(5, 3)) # 8

# Uso común: como argumento
puntos = [(1, 5), (3, 2)]
ordenados_y = sorted(puntos, key=lambda p: p[1])
print(ordenados_y) # [(3, 2), (1, 5)]

Útil para funciones simples usadas una sola vez.

`map()`

Aplica una función a CADA elemento de un iterable.

Sintaxis:
map(funcion, iterable, ...)

Devuelve un iterador (convertir a lista con `list()`).

Ejemplo:
nums = [1, 2, 3]
cuadrados_iter = map(lambda x: x**2, nums)

# Convertir a lista para ver
print(list(cuadrados_iter)) # [1, 4, 9]

nombres = ["ana", "eva"]
mayus_iter = map(str.upper, nombres)
print(list(mayus_iter)) # ['ANA', 'EVA']

`filter()`

Filtra elementos de un iterable según una función booleana.

Sintaxis:
filter(funcion_bool, iterable)

Devuelve un iterador (convertir a lista con `list()`).

Ejemplo:
nums = [1, 2, 3, 4, 5]
pares_iter = filter(lambda n: n % 2 == 0, nums)

# Convertir a lista para ver
print(list(pares_iter)) # [2, 4]

mezcla = [0, "OK", None, []]
truthy_iter = filter(None, mezcla) # Filtra falsy
print(list(truthy_iter)) # ['OK']

Comparación con List Comprehensions

Las comprensiones a menudo son más legibles para mapeos y filtros simples:

# Map equivalente
cuadrados = [x**2 for x in nums]
# Filter equivalente
pares = [n for n in nums if n % 2 == 0]

`map`/`filter` pueden ser útiles con funciones complejas o por eficiencia de memoria (iteradores).
