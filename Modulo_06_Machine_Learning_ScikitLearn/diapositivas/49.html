Módulo 6: Conceptos Clave de Machine Learning

Aprendiendo de los datos con Scikit-learn.

¿Qué es Machine Learning?

Es un subcampo de la Inteligencia Artificial que se enfoca en desarrollar sistemas que pueden aprender de los datos e identificar patrones para tomar decisiones o hacer predicciones con mínima intervención humana.

Scikit-learn

La biblioteca de ML más popular y completa para Python (excluyendo Deep Learning).

Ofrece:

  • Implementaciones eficientes de algoritmos de ML.
  • Herramientas para evaluación, selección de características y preprocesamiento.
  • Una API consistente y fácil de usar.

Instalación: pip install scikit-learn

Aprendizaje Supervisado

Aprende de ejemplos etiquetados (entradas + salidas conocidas).

Subtipos:

  • Clasificación: Salida es una categoría discreta (ej: spam/no spam).
  • Regresión: Salida es un valor numérico continuo (ej: precio de casa).

Aprendizaje No Supervisado

Encuentra patrones en datos sin etiquetas predefinidas.

Subtipos Comunes:

  • Clustering: Agrupar datos similares.
  • Reducción de Dimensionalidad: Reducir número de variables.
  • Detección de Anomalías: Identificar datos inusuales.

Terminología Clave

Dataset: Colección de datos.

Instancia/Muestra: Una fila individual.

Característica (Feature): Columna de entrada (X).

Etiqueta (Label)/Objetivo: Columna de salida a predecir (y).

Modelo: Algoritmo entrenado.

Entrenamiento: Proceso de aprendizaje del modelo.

Prueba/Evaluación: Medir rendimiento en datos no vistos.

Predicción: Usar modelo para generar salida.

Flujo de Trabajo Básico (Supervisado)

  1. Preparar los Datos (Limpiar, preprocesar).
  2. Dividir los Datos (Train/Test set).
  3. Elegir un Modelo.
  4. Entrenar el Modelo (.fit()).
  5. Hacer Predicciones (.predict()).
  6. Evaluar el Modelo (Métricas).
