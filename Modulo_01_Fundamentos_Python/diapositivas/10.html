OpDescEj (`x=5, y=8`)Res

`<=`Menor/Igual`y <= 8``True`
 
edad = 25
print(edad >= 18) # True

Lógicos

Combinan expresiones booleanas.

OpDescEj (`a=T, b=F`)Res
`and`Ambos `True``a and b``False`
`or`Al menos uno `True``a or b``True`
`not`Invierte valor`not a``False`
 
registrado = True
pago_ok = False
print(registrado and pago_ok) # False
print(registrado or pago_ok)  # True
print(not registrado)         # False
