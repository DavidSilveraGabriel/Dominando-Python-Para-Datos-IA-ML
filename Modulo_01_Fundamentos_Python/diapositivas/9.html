Módulo 1: Variables y Tipos de Datos Primitivos

Almacenando y clasificando información.

Asignación de Variables

Usamos `=` para asignar un valor a un nombre (etiqueta).

# nombre_variable = valor
mensaje = "Hola"
numero = 42
es_valido = True

print(mensaje) # Salida: Hola

# Puede cambiar
numero = 100
print(numero) # Salida: 100

Nombres de Variables (PEP 8)

  • Descriptivos: `edad_usuario` (no `x`).
  • `snake_case`: minúsculas, guion bajo (`_`).
  • No palabras reservadas (`if`, `class`...).
  • No empezar con número (`1var` no).
  • Sensible a mayúsculas (`VAR` != `var`).
  • Evitar caracteres especiales (excepto `_`).
# Buenos
contador_visitas = 0
precio_total = 99.95

# Malos (evitar)
# x = 0
# total$ = 99.95
# class = "Python"

Tipos de Datos Primitivos

Enteros (`int`)

Números sin decimales.

edad = 35 cantidad = -10

Flotantes (`float`)

Números con decimales.

precio = 19.99 pi = 3.14159

Booleanos (`bool`)

Verdadero o Falso.

es_mayor = True activo = False

Cadenas (`str`)

Texto entre comillas.

nombre = "Alice" saludo = 'Hola'

Verificando el Tipo (`type()`)

numero = 10
print(type(numero)) # Salida: 

texto = "Python"
print(type(texto))  # Salida:

Python tiene **tipado dinámico**: infiere el tipo automáticamente.

mi_var = 5      # Es int
mi_var = "Texto" # Ahora es str
