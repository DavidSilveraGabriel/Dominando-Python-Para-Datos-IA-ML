La Función `print()`

Muestra mensajes y valores en la salida.

print("Mensaje directo.")
nombre = "Ana"
edad = 30
print("Nombre:", nombre, "Edad:", edad)
print(f"f-string: {nombre} tiene {edad} años.")

Comentarios

Líneas ignoradas por Python, para explicaciones.

  • Una línea: Empieza con `#`.
  • Multilínea (Docstring): Con `"""` o `'''` (uso específico).
# Comentario de una línea
velocidad = 90 # Comentario al final

# print("No se ejecuta")

"""
Docstring multilínea, usado
para documentar funciones/clases.
"""

Sangría (Indentation) - ¡CRÍTICO!

Define la estructura y bloques de código en Python. ¡Reemplaza a `{}`!

  • Estándar: **4 espacios** por nivel.
  • Consistencia es clave. No mezclar espacios y tabs.
  • Errores de sangría (`IndentationError`) son comunes al inicio.
# CORRECTO
if puntaje > 50:
    # 4 espacios
    print("Aprobado!")
    print("Felicidades.")
print("Fin.") # Fuera del if

# INCORRECTO (IndentationError)
# if puntaje > 50:
# print("Error") # Falta sangría

# INCORRECTO (IndentationError)
# if puntaje > 50:
#     print("Nivel 1")
#        print("Nivel 2 erróneo") # Sangría inconsistente

¡La sangría es obligatoria y fundamental!
