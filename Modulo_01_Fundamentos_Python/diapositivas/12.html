Módulo 1: Control de Flujo - Condicionales

Tomando decisiones en tu código: `if`, `elif`, `else`.

`if` - Si...

Ejecuta un bloque si la condición es `True`.

if condicion:
    # Bloque si True (¡Sangría!)
    print("Condición Verdadera")
print("Siempre después")
edad = 20
if edad >= 18:
    print("Mayor de edad.") # Se ejecuta

`else` - Si no...

Ejecuta un bloque si la condición del `if` es `False`.

if condicion:
    # Bloque si True
else:
    # Bloque si False
    print("Condición Falsa")
print("Siempre después")
edad = 16
if edad >= 18:
    print("Mayor de edad.")
else:
    print("Menor de edad.") # Se ejecuta

`elif` - Si no, si...

Comprueba condiciones adicionales si las anteriores fueron `False`.

if cond1:
    # Bloque si cond1 True
elif cond2:
    # Bloque si cond1 False Y cond2 True
elif cond3:
    # Bloque si cond1,2 False Y cond3 True
else:
    # Bloque si NINGUNA es True
print("Siempre después")

Se ejecuta el primer bloque cuya condición sea `True` y se sale.

Ejemplo `if-elif-else`

nota = 78

if nota >= 90:
    print("Calificación: A")
elif nota >= 80:
    print("Calificación: B")
elif nota >= 70:
    print("Calificación: C") # Se ejecuta esta
elif nota >= 60:
    print("Calificación: D")
else:
    print("Calificación: F")
# Salida: Calificación: C

Condiciones Anidadas

Puedes poner `if/elif/else` dentro de otros.

edad = 25
tiene_lic = True

if edad >= 18:
    # Bloque externo
    print("Mayor de edad.")
    if tiene_lic:
        # Bloque interno
        print("Puede conducir.")
    else:
        print("Necesita licencia.")
else:
    print("Menor de edad.")

¡Los condicionales permiten que tus programas tomen decisiones lógicas!
