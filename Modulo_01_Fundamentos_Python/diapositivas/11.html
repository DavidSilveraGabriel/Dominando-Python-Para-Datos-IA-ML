Módulo 1: Estructuras de Datos

Organizando colecciones de información.

Listas (`list`)

Ordenadas, Mutables, Indexadas, Permiten duplicados.

# Creación
frutas = ["manzana", "pera", "manzana"]
print(frutas[0]) # manzana
print(frutas[-1])# manzana (último)

# Modificar
frutas[1] = "kiwi" # ['manzana', 'kiwi', 'manzana']

# Añadir
frutas.append("naranja") # Al final
frutas.insert(1, "uva") # En índice 1

# Eliminar
frutas.remove("manzana") # Primera ocurrencia
frutas.pop(0) # Por índice
del frutas[0]

# Otros
print(len(frutas)) # Longitud
print("kiwi" in frutas) # True/False

Tuplas (`tuple`)

Ordenadas, Inmutables, Indexadas, Permiten duplicados.

# Creación
coords = (10.5, -3.2)
colores = (255, 0, 128)
un_elem = (5,) # ¡Coma obligatoria!
vacia = ()

print(coords[0]) # 10.5
print(colores[-1]) # 128

# NO SE PUEDEN MODIFICAR
# coords[0] = 5.0 # TypeError!

# Otros
print(len(colores)) # 3
print(255 in colores) # True

# Usos: Coordenadas, registros fijos.

Diccionarios (`dict`)

Pares Clave-Valor, Mutables, Acceso por Clave Única, Ordenado por inserción (Python 3.7+).

# Creación
estudiante = {"nombre": "Carlos", "edad": 22}
vacio = {}

# Acceso por clave
print(estudiante["nombre"]) # Carlos
print(estudiante.get("edad")) # 22
print(estudiante.get("nota", "N/A")) # N/A

# Modificar / Añadir
estudiante["edad"] = 23
estudiante["curso"] = "Física"

# Eliminar
del estudiante["edad"]
curso = estudiante.pop("curso")

# Otros
print(list(estudiante.keys())) # ['nombre']
print(list(estudiante.values()))# ['Carlos']
print(list(estudiante.items())) # [('nombre', 'Carlos')]
print(len(estudiante)) # 1
print("nombre" in estudiante) # True (busca en claves)

Conjuntos (`set`)

Elementos Únicos, Mutables, Desordenados, No indexados.

# Creación
numeros = {1, 2, 3, 3, 4} # {1, 2, 3, 4}
letras = set("hola") # {'h', 'o', 'l', 'a'}
vacio = set() # ¡No usar {}!

# Añadir (ignora duplicados)
numeros.add(5)
numeros.add(2) # No hace nada

# Eliminar
numeros.remove(3) # Error si no existe
numeros.discard(1) # No da error si no existe
numeros.pop() # Elimina uno arbitrario

# Otros
print(len(numeros))
print(5 in numeros) # Muy rápido

# Operaciones
a = {1, 2}; b = {2, 3}
print(a.union(b)) # {1, 2, 3}
print(a.intersection(b)) # {2}
print(a.difference(b)) # {1}
