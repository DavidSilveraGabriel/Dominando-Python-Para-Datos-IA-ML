Módulo 1: Control de Flujo - Bucles

Repitiendo acciones: `for`, `while`.

Bucle `for` - Iterar sobre Secuencias

Ejecuta un bloque para cada elemento de una lista, tupla, string, `range`, etc.

for variable in secuencia:
    # Bloque a repetir
    print(variable)
# Sobre lista
frutas = ["a", "b"]
for f in frutas:
    print(f) # a, luego b
# Sobre range
for i in range(3): # 0, 1, 2
    print(i)
# Sobre string
for letra in "Hi":
    print(letra) # H, luego i
# Sobre dict (claves)
d = {"k1":1, "k2":2}
for k in d:
    print(k) # k1, luego k2

Bucle `while` - Mientras la Condición sea Verdadera

Ejecuta un bloque repetidamente mientras una condición sea `True`.

while condicion:
    # Bloque a repetir
    print("Aún True")
    # ¡ALGO debe hacer False la condición!

¡Cuidado con los bucles infinitos si la condición nunca cambia a `False`!

contador = 0
while contador < 3:
    print(f"Cont: {contador}")
    contador += 1 # Clave para terminar
# Salida: Cont: 0, Cont: 1, Cont: 2

Controlando Bucles: `break` y `continue`

`break`

Termina el bucle actual inmediatamente.

for i in range(10):
    if i == 5:
        print("Encontrado 5, saliendo.")
        break
    print(i) # Imprime 0, 1, 2, 3, 4

`continue`

Salta al inicio de la siguiente iteración.

for i in range(5):
    if i % 2 == 0: # Si es par
        continue # Salta el print
    print(f"Impar: {i}")
# Salida: Impar: 1, Impar: 3
