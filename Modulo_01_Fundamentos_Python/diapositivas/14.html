Módulo 1: Funciones

Bloques de código reutilizables y organizados.

Definición (`def`)

def nombre_funcion(param1, param2):
    """Docstring opcional."""
    # Cuerpo de la función
    resultado = param1 + param2
    return resultado # Opcional

Llamada

Ejecutar la función por su nombre.

# Definición
def saludar(nombre):
    print(f"Hola, {nombre}")

# Llamada
saludar("Mundo") # Salida: Hola, Mundo

def sumar(a, b): return a + b
total = sumar(5, 3) # total = 8
print(total)

Parámetros y Argumentos

Parámetros: Variables en la definición.
Argumentos: Valores pasados en la llamada.

Tipos de Argumentos:

  • Posicionales: Por orden. `func(1, 2)`
  • Palabra Clave: Por nombre. `func(b=2, a=1)`
def describir(animal, nombre):
    print(f"{animal} llamado {nombre}")

describir("perro", "Fido") # Posicional
describir(nombre="Mishi", animal="gato") # Keyword

Valores por Defecto

Asigna un valor en la definición si no se pasa argumento.

def potencia(base, exponente=2):
    return base ** exponente

print(potencia(5))    # 25 (usa defecto exp=2)
print(potencia(5, 3)) # 125 (usa exp=3)

Params con defecto deben ir al final.

Retorno (`return`)

Envía un valor de vuelta. Termina la función.

def dividir(a, b):
    if b == 0:
        return None # Devuelve None en error
    return a / b

res = dividir(10, 2) # res = 5.0
err = dividir(5, 0)  # err = None

Si no hay `return`, devuelve `None` implícitamente.

Alcance (Scope)

Variables Locales

Definidas DENTRO de la función (incl. parámetros). Solo existen en la función.

Variables Globales

Definidas FUERA de cualquier función. Se pueden LEER desde dentro.

Para MODIFICAR una global desde dentro, usar `global` (¡evitar si es posible!).

g_var = "Global"

def mi_func():
    l_var = "Local"
    print(l_var) # OK
    print(g_var) # OK (lectura)

mi_func()
print(g_var) # OK
# print(l_var) # Error! No existe fuera.

count = 0
def incrementar():
    global count # Aviso: modifico global
    count += 1

incrementar() # count es 1

¡Las funciones hacen tu código modular, reutilizable y más fácil de leer!
