Módulo 7: Introducción a las Redes Neuronales

Los bloques de construcción del Deep Learning.

¿Qué es una Red Neuronal Artificial?

Un modelo computacional inspirado en la estructura y funcionamiento del cerebro humano. Consiste en nodos interconectados ("neuronas") organizados en capas.

Inspiración Biológica

Imitan la forma en que las neuronas biológicas procesan información: reciben señales, las combinan y, si la suma supera un umbral, activan una señal de salida.

Neuronas Artificiales (Perceptrones)

Unidad básica de una red neuronal. Recibe entradas, les asigna pesos, suma las entradas ponderadas y pasa el resultado a través de una función de activación para producir una salida.

`Salida = FunciónActivacion(Suma(Entradas * Pesos) + Bias)`

Funciones de Activación

Determinan la salida de una neurona. Introducen **no linealidad** en la red, permitiéndole aprender patrones complejos.

  • **Sigmoide:** Comprime la salida entre 0 y 1.
  • **ReLU (Rectified Linear Unit):** Salida 0 para entradas negativas, la entrada misma para entradas positivas. Muy común.
  • **Tanh (Tangente Hiperbólica):** Comprime la salida entre -1 y 1.

Capas

Las neuronas se organizan en capas:

  • **Capa de Entrada:** Recibe las características del dataset.
  • **Capas Ocultas:** Capas intermedias donde se realiza la mayor parte del procesamiento.
  • **Capa de Salida:** Produce la predicción final del modelo.

Una red con más de una capa oculta es una red de **Deep Learning**.

Las redes neuronales aprenden ajustando los pesos y biases de las conexiones entre neuronas.
