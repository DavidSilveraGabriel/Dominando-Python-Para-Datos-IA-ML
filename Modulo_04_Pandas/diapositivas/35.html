Módulo 4: Pandas - DataFrame

La estructura tabular 2D esencial de Pandas.

¿Qué es un DataFrame?

Estructura bidimensional tabular (filas y columnas) con etiquetas.

Componentes: Datos + Índice (filas) + Columnas (etiquetas).

Creando DataFrames

1. Desde Dict de Listas/Arrays

datos = {'ColA': [1,2], 'ColB': [3,4]}
df1 = pd.DataFrame(datos)

2. Desde Array NumPy 2D

arr = np.arange(6).reshape(3,2)
df2 = pd.DataFrame(arr,
        index=['r1','r2','r3'],
        columns=['C1','C2'])

3. Desde Lista de Dicts

lista_d = [{'a':1, 'b':2}, {'a':3, 'b':4, 'c':5}]
df3 = pd.DataFrame(lista_d)
# NaN para 'c' en la primera fila

Exploración Básica

Estadísticas descriptivas.

Tupla (filas, columnas).

df.dtypes

Tipos de dato por columna.

df.index

Índice de filas.

df.columns

Índice de columnas.

df.values

Datos como array NumPy.

Selección Básica

df['NombreCol']

Selecciona una columna (devuelve Serie).

df[['Col1', 'Col2']]

Selecciona múltiples columnas (devuelve DataFrame).

df[inicio:fin]

Slice de filas por posición (¡solo funciona directamente para filas!).

Para selección robusta por filas/columnas, usa `.loc[]` (etiquetas) y `.iloc[]` (posición).
