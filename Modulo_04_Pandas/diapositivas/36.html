Módulo 4: Lectura y Escritura de Datos

Importando y exportando datos con Pandas (CSV, Excel...).

Archivos CSV

Leer: `pd.read_csv()`

df = pd.read_csv('archivo.csv')

Params comunes: `sep`, `header`, `names`, `index_col`, `usecols`, `dtype`, `na_values`, `skiprows`, `nrows`.

# Ejemplo: Delimitador ';', sin header
df_ej = pd.read_csv('datos.csv', sep=';',
                    header=None, names=['A','B'])

Escribir: `df.to_csv()`

df.to_csv('salida.csv', index=False)

Params comunes: `sep`, `index` (usualmente `False`), `header`, `columns`, `mode` (`'w'`, `'a'`), `encoding`, `float_format`.

# Ejemplo: Separador tab, sin índice
df.to_csv('output.tsv', sep='\\t', index=False)

Archivos Excel

Requiere `openpyxl` (para `.xlsx`) y/o `xlrd`.

Leer: `pd.read_excel()`

df = pd.read_excel('archivo.xlsx')

Params comunes: `sheet_name` (nombre, índice, lista o `None` para todas), `header`, `names`, `index_col`, `usecols`, `dtype`, `skiprows`, `nrows`.

# Ejemplo: Leer hoja 'Datos', usar col 0 como índice
df_ej = pd.read_excel('reporte.xlsx',
                      sheet_name='Datos', index_col=0)

Escribir: `df.to_excel()`

df.to_excel('salida.xlsx', sheet_name='Resultados', index=False)

Params comunes: `excel_writer` (ruta o `ExcelWriter`), `sheet_name`, `index`, `header`, `columns`, `startrow`, `startcol`.

# Ejemplo: Escribir múltiples hojas
with pd.ExcelWriter('output.xlsx') as writer:
    df1.to_excel(writer, sheet_name='Hoja1')
    df2.to_excel(writer, sheet_name='Hoja2')
