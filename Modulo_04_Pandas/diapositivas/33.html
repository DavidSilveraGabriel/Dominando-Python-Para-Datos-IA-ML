Módulo 4: Introducción a Pandas: Series y DataFrames

La biblioteca esencial para manipulación y análisis de datos.

¿Qué es Pandas?

Biblioteca open-source para estructuras de datos de alto rendimiento y análisis de datos.

Tareas Fundamentales:

  • Carga y guardado de datos (CSV, Excel, SQL...)
  • Limpieza y preparación (nulos, duplicados...)
  • Exploración y análisis (selección, filtrado...)
  • Agrupación y agregación
  • Combinación de datasets
  • Manejo de series temporales

Construido sobre NumPy, añadiendo índices/etiquetas.

Importando Pandas

Convención estándar: alias `pd`.

import pandas as pd
import numpy as np # Usualmente juntos

(Instalar: `pip install pandas` o `conda install pandas`)

Estructuras Principales

1. Series

Array 1D etiquetado (como una columna).

Componentes: Valores + Índice (etiquetas).

2. DataFrame

Estructura 2D tabular etiquetada (como tabla/hoja de cálculo).

Componentes: Datos + Índice (filas) + Columnas (etiquetas de columna).

Pandas facilita trabajar con datos estructurados de forma intuitiva.
