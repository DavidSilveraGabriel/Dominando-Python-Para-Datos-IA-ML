Personalización Matplotlib (Avanzado)

Colores, Líneas y Marcadores

Args en `plot()`/`scatter()`: `color` (nombre, hex, RGB), `linestyle`/`ls` (`'-'`, `'--'`, `':'`, `'-.'`), `linewidth`/`lw`, `marker` (`'o'`, `'x'`, `'+'`, `'s'`, ...), `markersize`/`ms`, etc.

ax.plot(x, y, color='magenta', ls=':', lw=2, marker='D')
ax.plot(x, y2, 'go--') # Formato corto: verde, círculo, discontinuo

Límites, Texto y Anotaciones

  • Límites: `ax.set_xlim(min, max)`, `ax.set_ylim(min, max)`, `ax.axis([xmin, xmax, ymin, ymax])`.
  • Texto: `ax.text(x, y, 'texto', ...)`
  • Anotaciones: `ax.annotate('texto', xy=(px,py), xytext=(tx,ty), arrowprops=...)`

Escalas y Guardado

  • Escala Log: `ax.set_xscale('log')` / `yscale`.
  • Guardar: `fig.savefig('nombre.png', dpi=300, bbox_inches='tight')`.

Introducción a Seaborn

¿Qué es Seaborn?

Biblioteca de visualización basada en Matplotlib.

  • Interfaz de alto nivel para gráficos estadísticos.
  • Integración excelente con Pandas DataFrames.
  • Estilos y paletas de colores atractivos por defecto.
  • Simplifica gráficos complejos (heatmaps, violinplots...).

Importación y Tema

import seaborn as sns
import matplotlib.pyplot as plt

# Aplicar un tema visual (opcional)
sns.set_theme(style="whitegrid")
# Otros: "darkgrid", "white", "ticks"...
