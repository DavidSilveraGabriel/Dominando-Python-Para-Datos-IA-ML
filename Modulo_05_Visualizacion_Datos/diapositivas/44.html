Módulo 5: Introducción a Matplotlib: Gráficos Básicos

Visualizando datos con la biblioteca fundamental de Python.

¿Qué es Matplotlib?

Biblioteca base para visualización estática, animada e interactiva.

  • Control detallado sobre cada elemento.
  • Interfaz común: `matplotlib.pyplot` (similar a MATLAB).
  • Base para otras bibliotecas (Seaborn).

Importación Estándar 

import matplotlib.pyplot as plt
import numpy as np
import pandas as pd # Opcional

(Instalar: `pip install matplotlib`)

Gráfico de Líneas (`plt.plot`) 

# Datos
x = np.linspace(0, 10, 100)
y = np.sin(x)

# Crear y configurar gráfico
plt.figure(figsize=(8, 3)) # Tamaño opcional
plt.plot(x, y, label='Seno(x)') # Dibuja la línea

plt.title("Título del Gráfico")
plt.xlabel("Etiqueta Eje X")
plt.ylabel("Etiqueta Eje Y")
plt.legend() # Muestra label
plt.grid(True) # Cuadrícula
