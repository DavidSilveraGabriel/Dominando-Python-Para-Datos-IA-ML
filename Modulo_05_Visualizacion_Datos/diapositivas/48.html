Módulo 5: (Opcional) Gráficos Interactivos

Explorando datos dinámicamente con Plotly y Streamlit.

Plotly / Plotly Express

Biblioteca para gráficos interactivos (zoom, pan, hover) y de calidad publicación.

Plotly Express (`px`):

Interfaz de alto nivel (similar a Seaborn) para creación rápida.

import plotly.express as px

# df es un DataFrame de Pandas
fig = px.scatter(df, x='col_x', y='col_y',
                 color='categoria', size='tamaño',
                 hover_data=['col_extra'],
                 title="Gráfico Plotly")

# Muestra en navegador o notebook
fig.show()

Genera HTML interactivo. Amplia gama de gráficos (3D, mapas...).

Streamlit

Framework para crear y compartir apps web de datos fácil y rápidamente.

Características:

  • Convierte scripts Python en apps web.
  • Widgets interactivos (sliders, botones, selectores...).
  • Integra gráficos (Plotly, Matplotlib, etc.).
  • Ideal para dashboards, prototipos, compartir resultados.
Ver Ejemplo Básico de Código (`mi_app.py`) 
import streamlit as st
import pandas as pd
import plotly.express as px

st.title("Mi App Streamlit")

# Cargar datos (ej. con @st.cache_data)
df = pd.read_csv(...)

# Widgets
valor_slider = st.slider("Filtro:", ...)
opciones = st.multiselect("Categorías:", ...)

# Lógica y filtrado
df_filtrado = df[...]

# Mostrar datos y gráficos
st.dataframe(df_filtrado)
fig = px.scatter(df_filtrado, ...)
st.plotly_chart(fig) # Mostrar Plotly
# st.pyplot(fig_mpl) # Mostrar Matplotlib
Ejecutar:
$ streamlit run mi_app.py

Plotly y Streamlit llevan tus visualizaciones y análisis a un nivel interactivo y compartible.
