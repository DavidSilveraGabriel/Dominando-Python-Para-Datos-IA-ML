Módulo 5: Más Gráficos Básicos y Enfoque OO

Dispersión (`plt.scatter`)

Relación entre 2 variables numéricas.

# Datos aleatorios
x = np.random.rand(30)
y = x + np.random.randn(30)*0.1

plt.figure(figsize=(5,3)) # Pequeño
plt.scatter(x, y, color='red', marker='o')
plt.title("Dispersión")
plt.xlabel("X")
plt.ylabel("Y")
plt.grid(True, alpha=0.5)
# plt.show()

Barras (`plt.bar`)

Comparar cantidades entre categorías.

categorias = ['A', 'B', 'C']
valores = [15, 28, 22]

plt.figure(figsize=(5,3))
plt.bar(categorias, valores, color='skyblue')
plt.title("Barras")
plt.xlabel("Categoría")
plt.ylabel("Valor")
# plt.show()

Histograma (`plt.hist`)

Distribución de 1 variable numérica (frecuencia por 'bins').

datos = np.random.randn(500)

plt.figure(figsize=(5,3))
plt.hist(datos, bins=25,
         color='lightgreen',
         edgecolor='black')
plt.title("Histograma")
plt.xlabel("Valor")
plt.ylabel("Frecuencia")
# plt.show()

Enfoque Orientado a Objetos (Recomendado)

Más control y flexibilidad, especialmente para gráficos complejos o subplots.

Pasos:

  1. Crear `Figure` y `Axes` explícitamente (`plt.subplots()`).
  2. Llamar métodos sobre el objeto `Axes` (`ax.plot()`, `ax.scatter()`, etc.).
  3. Configurar usando métodos de `ax` (`ax.set_title()`, `ax.set_xlabel()`, etc.).

Más verboso, pero más potente.

# Datos
x = np.linspace(0, 5, 50)
y = x**2

# 1. Crear Figura y Ejes
fig, ax = plt.subplots(figsize=(6, 3))

# 2. Usar métodos de 'ax'
ax.plot(x, y, label='x^2', color='purple')

# 3. Configurar con 'ax'
ax.set_title("Gráfico OO")
ax.set_xlabel("X")
ax.set_ylabel("Y = X^2")
ax.legend()
ax.grid(True)
